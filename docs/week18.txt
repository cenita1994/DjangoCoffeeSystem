Step 8: DjangoCoffeeSystem/settings.py
Hanapin ito:

Palitan ng:

'APP_DIRS': True,


Step 9: inventory/views.py

from django.shortcuts import render

def home(request):
    return render(request, 'home.html')

def product(request):
    return render(request, 'inventory/product.html')

def about(request):
    return render(request, 'inventory/about.html')

Step 12: DjangoCoffeeSystem/settings.py

Hanapin ito:

STATIC_URL = 'static/'

So magiging ganito:

STATIC_URL = 'static/'

STATICFILES_DIRS = [
    BASE_DIR / 'static',
]
